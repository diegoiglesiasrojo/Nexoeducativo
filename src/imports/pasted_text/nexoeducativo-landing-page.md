Design a modern, high-converting, single-page landing page for an academic training brand called “Nexoeducativo”.

Context:
- This website sells academic training courses.
- The business goal is lead generation and course sales through WhatsApp.
- There is no backend, no login, no cart, no checkout, and no multi-page flow.
- The page must be designed to later be implemented with Next.js, React, TypeScript, Tailwind CSS, and reusable components.
- Use a mobile-first approach and fully responsive design.
- The website language is Spanish.

Brand:
- Use the attached Nexoeducativo logo in the header and footer.
- Build the visual identity around the logo style: professional, educational, trustworthy, modern, clean.
- Use a premium but friendly aesthetic.
- Prioritize clarity, trust, readability, and conversion.

Main objective:
- Convince visitors to contact via WhatsApp and inquire about / enroll in academic courses.

Target audience:
- Adults interested in academic training, teacher training, pedagogical qualification, diploma programs, educational updates, and professional development.
- Users should feel trust, academic seriousness, and ease of enrollment.

Color modes:
- Include both Light Mode and Dark Mode versions of the landing page.
- The design must work well in both themes.
- Keep strong readability, accessible contrast, and visual consistency in both modes.
- The dark mode should feel elegant, modern, and professional, not overly saturated.
- The light mode should feel clean, academic, and trustworthy.

Create a one-page landing page with these sections in this exact order:

1. Responsive header
- Sticky header
- Brand logo on the left
- Navigation menu with anchor links to sections
- On mobile: hamburger menu / dropdown navigation
- On desktop: horizontal navigation bar, no dropdown
- Include a primary CTA button: “Consultar por WhatsApp”
- Include a theme toggle for Light Mode / Dark Mode

2. Hero section
- Strong headline focused on academic growth and enrollment
- Supporting text explaining the value proposition
- Primary CTA button to WhatsApp
- Secondary CTA button like “Ver cursos” that scrolls to the courses section
- Include a professional educational visual or illustration area
- Make the hero visually impactful but clean

3. Image carousel / slider
- A responsive carousel with promotional academic/course images
- Large rounded cards or banners
- Mobile-friendly swipe feel
- Desktop version should feel polished and premium

4. Featured courses / programs section
- Show course cards for examples like:
  - Diplomaturas
  - Actualizaciones académicas
  - Tramo pedagógico
  - Formación docente
- Each card should include title, short description, benefits, and CTA button to WhatsApp
- Use reusable card design
- Grid on desktop, stacked cards on mobile

5. Benefits / value proposition section
- Explain why users should choose Nexoeducativo
- Example bullets/cards:
  - Aprendizaje flexible
  - Acompañamiento académico
  - Contenido práctico y actualizado
  - Crecimiento profesional
  - Inscripción sencilla
- Use icons or visual highlights

6. Testimonials section
- Include a responsive testimonials layout
- Show at least 3 testimonial cards
- Include student/client name, short role/profile, and quote
- Design for trust and social proof
- Make it visually distinct from the rest of the page

7. FAQ section
- Accordion-style frequently asked questions
- Questions related to modality, course validity, enrollment, support, and certificates
- Mobile-first interaction design

8. Contact / final CTA section
- Strong closing message
- Final WhatsApp CTA button
- Short text encouraging immediate contact
- Include a compact contact information block

9. Footer
- Typical footer structure
- Brand/logo
- Navigation links
- Contact information
- Social placeholders
- Copyright
- Clean professional layout

Persistent conversion element:
- Add a floating WhatsApp button fixed to the bottom-right corner on all screen sizes
- It must always remain visible and clearly clickable
- Use WhatsApp-style affordance but keep visual harmony with the brand

Design requirements:
- Mobile first
- Fully responsive
- Clear spacing and hierarchy
- Large readable typography
- Clean section separation
- Rounded corners
- Soft shadows
- Modern layout system
- Professional landing page aesthetics
- Accessible contrast and readable text sizes

Important constraints:
- Do NOT design an e-commerce store
- Do NOT add cart, checkout, user account, or product filters
- Do NOT make it look like a marketplace
- Keep it as a sales landing page focused on one conversion goal: WhatsApp contact
- Keep everything in a single page with anchor navigation

Implementation direction:
- Structure the design so it can later be implemented with Next.js + React + TypeScript + Tailwind CSS
- Use a component-based layout easy to translate into reusable frontend sections
- Keep the UI clean and realistic for production

Deliver the design as a structured UI that is easy to translate into components:
- Header
- Hero
- Carousel
- Course cards
- Benefits section
- Testimonials
- FAQ
- Contact CTA
- Footer
- Floating WhatsApp button
- Theme toggle
